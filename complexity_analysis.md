# Complexity Analysis

## Time Complexity

- **Bubble Sort**: O(n²)
- **Insertion Sort**: O(n²)
- **Merge Sort**: O(n log n)
- **Linear Search**: O(n)
- **Binary Search**: O(log n)

## Space Complexity

- **Bubble Sort**: O(1)
- **Insertion Sort**: O(1)
- **Merge Sort**: O(n)
- **Linear Search**: O(1)
- **Binary Search**: O(1)
